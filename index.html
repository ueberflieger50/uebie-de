    Moodle Tools

    Dies bezieht sich auf zwei Tools die minie4 gebaut hat, um von der Online-Lernplattform der Schule Daten zu besorgen und anzuzeigen.
    1. Hier wird die API der Plattform, die die Schule nutzt angefragt um die Daten in einem neuen UI anzuzeigen.
    2. Hierbei wird durch ein automatisiertes Script der Wochenplan abgerufen, um so ein regelmäßigen log zu haben, was sich verändert hat.
